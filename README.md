Auto move by pressing a hotkey for Stalker Anomaly. Compatible with GAMMA. Configurable with MCM.

Default auto move key is `mouse button 4`.

## Install

1) Download latest [release](https://github.com/FaithBeam/stalker-anomaly-auto-move/releases/latest)
2) Open ModOrganizer
3) Drag archive at end of list
4) Click through prompts

## Rebind Key

1) Open Auto Move MCM menu
    - Mod Configuration Menu > Auto Move
2) Click `Auto Move - Activate Key`
3) Press whatever key you want to activate auto move
4) Save
